# PSYCHO — Horror Quest

**Face your fears. Book your nightmare.**

PSYCHO is a multilingual (EN/RO/RU) horror quest booking website. Browse immersive escape rooms with live actors, check pricing, and book your session online — all from a fast, mobile-first static site with zero backend.

![Screenshot](screenshot.png)

## Live Demo

**[psycho-phi.vercel.app](https://psycho-phi.vercel.app)**

## Features

- 3 horror quest rooms with fear ratings
- Online booking form with email notifications (FormSubmit.co)
- Flatpickr date picker + time slot selection
- Multilingual: English, Romanian, Russian
- Dark/light theme with system preference detection
- Booking history saved in localStorage
- Mobile-first responsive design
- Google Maps embed
- Full SEO meta tags + Open Graph

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- Zero frameworks, zero build tools
- CDN-only dependencies (Google Fonts, Flatpickr)
- FormSubmit.co for email (no API keys)

## Setup

1. Replace `YOUR_EMAIL@example.com` in the booking form action with your real email
2. Replace `YOURDOMAIN` in the Plausible script with your domain
3. Update contact info (phone, address, social links)
4. Deploy anywhere (Vercel, GitHub Pages, Netlify)
